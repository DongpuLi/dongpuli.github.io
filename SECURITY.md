# Security Policy

## Supported Version

The latest version on the `main` branch is the only actively maintained version.

Older commits and releases may not receive security updates.

## Reporting a Security Issue

If you discover a security issue related to this project, please:

- Open a GitHub Issue if the report does not disclose sensitive information.
- If the issue involves a potential security risk that should not be publicly disclosed, please contact the maintainer through GitHub before publishing details.

I will acknowledge reports as soon as practical and investigate confirmed issues promptly.

Thank you for helping improve the security and reliability of this project.
